# 🎋 あみだくじメーカー

ブラウザ上で動く、ドラマチック演出付きあみだくじアプリです。
フレームワーク不使用（HTML + CSS + Vanilla JS + Canvas）なので、`index.html` を開くだけで動作します。GitHub Pages でもそのまま公開可能です。

## 機能

### 1. くじの本数：2〜20本
スライダーで自由に選択できます。

### 2. あたりの種類と数を自由に設定
- 賞の「名前」「本数」「演出」を行単位で自由に追加・削除可能
  - 例：大吉×1（豪華宝箱）、中吉×2（紙吹雪）、凶×1（爆発）
- 残りの枠は「デフォルト賞」（例：小吉）で自動的に埋まります
- 賞の合計がくじの本数を超えるとエラー表示

### 3. ドラマチックな結果発表
- 参加者名を上のボタンでタップすると、光る玉があみだを緩急つけてたどるアニメーション
- ゴール後、ドラムロールでためてから結果をドーンと発表
- 演出は5種類（効果音付き・Web Audio APIによるシンセ生成）：

| 演出 | 内容 |
|---|---|
| 🏆 大当たり | 超絶豪華レア宝箱（金の光線・宝箱・キラキラ・金吹雪・ファンファーレ） |
| 🎉 あたり | カラフル紙吹雪 |
| 😊 普通 | ふわっと花が舞う控えめ演出 |
| 😞 ハズレ | 暗雲・雨・しょんぼり顔・下降トロンボーン音 |
| 💣 大外れ | フラッシュ→爆発→破片飛散→煙、画面シェイク付き爆発オチ |

### 4. 結果まとめ表示
「📋 結果をまとめて表示」ボタンで、全員分の結果を一覧表＋あみだ全体図で即時確認できます。

### オプション
- 参加者名の入力（未入力は「プレイヤーN」）
- アニメ速度（ゆっくり／普通／速い）
- 横線の密度（少なめ／普通／多め）
- 効果音 ON/OFF

## 使い方

```bash
# ローカルで動かす場合
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

またはそのまま `index.html` をブラウザで開いてください。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面構造（設定画面・ゲーム画面・結果モーダル） |
| `style.css` | スタイル＆エフェクト用アニメーション |
| `app.js` | あみだ生成・Canvas描画・経路アニメーション・まとめ表示 |
| `effects.js` | 5種類の演出エフェクト＋効果音（Web Audio） |
